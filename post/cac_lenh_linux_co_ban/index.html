Các Lệnh Linux Cơ Bản

Posted by     ng Ba.o on Friday, October 9, 2020

Các lệnh cơ bản trong Linux - Linux Commands

hello mn, hnay mình sẽ giới thiệu các câu lệnh cơ bản để làm việc trong hệ điều hành linux.

1. PWD COMMAND

Check đường dẫn thư mục hiện tại

2. MKDIR

Tạo Thư mục mới

ex : mkdir workspace
3. TOUCH

Tạo file mới

ex : touch index.php
4. LS

hiện thị các file ở thư mục hiện tại

5. LS -LS AND LL

hiện thị tất cả các file ở thư mục hiện tại kể cả file ẩn và quyền của file

6. CD

chuyển hướng tới file làm việc

  • cd .. chuyển lên 1 cấp thư mục trên
  • cd ~ chuyển đến home
  • cd + đường dẫn : chuyển đến file làm việc ( đường dẫn phải tồn tại trong thư mục hiện tại )
ex : cd /home/username/workspace
7. CAT

xem nội dung của file

ex: cat index.php
8. HEAD

xem dòng đầu tiên của file được chỉ định

ex: head index.php
9. Tail

xem dòng cuối cùng của file được chỉ định

ex: tail index.php
10. CP

copy file vào thư mục làm việc khác

ex: cp index.php workspace
11. MV

di chuyển file vào thư mục làm việc khác

ex: mv index.php workspace
12. RM

cho phép bạn xóa 1 file nếu có quyền

ex: rm index.php
13. RMDIR

cho phép bạn xóa 1 thư mục trống

ex: rmdir workspace
14. RM -RF

cho phép bạn xóa 1 thư mục được chỉ định nếu có đủ quền

ex: rm -rf workspace

trường hợp xóa file, folder nhưng không đủ quền thì mình phân lại quyền khoặc thêm lệnh sudo trước lệnh xóa

15. subl + đường dẫn file

mở file với subl (máy phải cài subl sẵn)

ex: subl index.php
16. code + đường dẫn file

mở file với visualcode (máy phải cài visual và config một ít)

ex: code index.php

PHÍM TẮT CƠ BẢN

CTRL + ALT + T

mở terminal

CTRL + A

nhảy về đầu dòng lệnh

CTRL + U

lưu lại lệnh đang nhập để thực thi lệnh khác

CTRL + Y

sử dụng lệnh vừa CTRL + U

CTRL + Q

tắt ứng dụng hiện tại

Tab

hỗ trợ gợi ý gõ lệnh nhanh hơn.
