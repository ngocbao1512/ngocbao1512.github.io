Đẩy Code Lên GitHub

Posted by     bao on Tuesday, October 12, 2021

Giới Thiệu

Có nhiều bạn mới học lập trình được giáo viên yêu câù nộp bài tập qua github, hay các bạn cuối kì làm xong bài tập đồ án. muốn lưu lại code, để nhỡ sau muốn học lại học cải thiện. nên hôm nay mình sẽ hướng dẫn các bạn cách đẩy code lên github

Giới thiệu qua về github thì nó là một hệ thống quản lý dự án và code, hoạt động giống như một mạng xã hội cho lập trình viên.

Thực Hiện

  • Bước 1 : trước tiên đối với các bạn dùng window mà chưa cài git, hãy download theo link sau: Download Git

    sau khi down load xong bạn có thể kiểm tra bằng lệnh

    git --version

  • Bước 2 : tạo 1 tài khoản github

  • Bước 3 : tạo 1 repository mới trên github click vào dấu cộng bên phải trên thanh ba trên cùng. chọn New repository -> nhập Repository Name(tên project của bạn), Description, chọn chế độ Public or Private -> Create Repository

  • Bước 4 : clone repository về máy

    bạn copy link url của repository vừa tạo và dùng lệnh sau để clone về máy

    git clone <link repo>

    vd

    git clone https://github.com/ngocbao1512/CSDL-QUAN_LY_THUOC.git

  • Bước 5 : Thêm Sửa xóa file clone xong thì bạn sẽ có một folder giống với tên repo mà bạn đã tạo, lúc này bạn thêm các file, folder cần lưu vào folder mới này.

  • Bước 6 : Push Code bạn mở terminal lên và cd đến folder mình clone về, hoặc mở visual studio code rồi mở thư mục mình clone về và bật terminal trong visual studio code. và bắt đầu nhập lệnh.

    git config --global user.name <username>
git config --global user.email <mailaddress>

    sau khi config username và email thì bạn nhập tiếp lệnh này để push code lên

    git add .
git commit -m "message commit"
git push origin master

đến bước này, thì bạn đã hoàn thành đẩy code lên github thành công. bạn lên github vào repo của bạn check lại. chúc các bạn thành công !..
